# AvarTranslator
A web-based morphological analyzer for the Avar language (a Northeast Caucasian language spoken in Dagestan, Russia). This tool provides automated analysis of Avar words and phrases, identifying roots, affixes, and grammatical features.
# Features
Morphological parsing of Avar text (lemmatization, affix segmentation, grammatical tagging).

Interactive web interface with a simple form for input and display of analysis results.

REST API endpoint (/analyzer/api/analyze/) for programmatic access.

Admin panel for managing linguistic data (roots, affixes, translations) via Django admin.
